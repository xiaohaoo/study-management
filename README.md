# 学习计划管理APP

Android课设小项目，用于对学习计划以及学习记录的管理，一条学习计划包含多条学习记录等。

## 开发环境
Android Studio v4.0.1

Gradle v6.1.1

Android Gradle Plugin v4.0.1

JDK 14

Android SDK 29

SQLite
